E-commerce - Orinoco

Présentation :

Orinoco est un site e-commerce qui créer plusieurs applications thématiques vendant qu'un seul groupe de produit par appli. Il s'agit ici de l'application web Orinours qui vend uniquement des ours en peluche.
Cette appli permet de choisir un ours en peluche, de sélectionner une couleur, de choisir une quantité et de l'ajouter à son panier. Le client peut ensuite modifier son panier en supprimant un article ou le vider complètement. Enfin, il peut remplir un formulaire de commande, après lequel il reçoit un message de confirmation avec un numéro de commande.

Objectifs :

- Créer le web design et le responsive en Css.
- Interagir avec le web service avec JavaScript (requêtes http, json).
- Valider des données de formulaire de contact avec javascript.
- Créer des évenements javascript (ajout/suppression d'articles, validation de commande...).
- Gérer le localstorage pour le panier client.
- Créer un plan de tests pour l'application.
