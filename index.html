Portfolio - OnePage

Présentation :

Le portfolio sur lequel vous êtes, est un OnePage qui permet de montrer mes projets, ma présentation et mes contacts.

Caractéristiques :

- Créer le web design et le responsive du OnePage en Pure Css avec Sass.
- Créer une animation néon pour le hero.
- Créer un carousel de présentation pour les projets.
- Créer un modal pop-up pour les fiches des projets.
- Création des vidéos de démonstration pour chaque projet.

Réseau social - Groupomania

Présentation :

Groupomania est un réseau social interne d'entreprise qui permet de favoriser les échanges entre collègues. Cette application web permet aux utilisateurs de créer un compte, de poster des messages gifs et d'écrire des commentaires. Elle contient aussi un compte admin pour la modération des messages des utilisateurs.

Caractéristiques :

- Créer l'API REST avec Node.js et Express.js.
- Personnaliser le contenu envoyé à un client web avec Multer.
- Authentifier une connexion sécurisée de l'utilisateur et maintenir sa session avec Bcrypt et Json web token
- Implémenter et gérer le stockage de données sécurisées à l'aide de Mysql et Sequelize.
- Créer le web design et le responsive de l'application avec le framework Vue.js et Bootstrap.

A PROPOS

Je m'appelle Laetitia, je suis développeur web fullstack ascendant frontend.

J'aime créer des sites web avec un design attractif pour les entreprises, les commerces et les indépendants comme moi.
